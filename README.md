# My Spotify Clone

A web-based music player mimicking the interface of Spotify, built with HTML, CSS, and JavaScript.

## Features

- **Music Playback**: Play, pause, and seek through tracks.
- **Navigation**: Skip to the next or previous song in the playlist.
- **Library View**: Displays a list of available songs.
- **Dynamic Controls**: Real-time track duration and current time display.
- **Responsive Design**: Adapts to different screen sizes.

## Technologies Used

- **HTML5**: For structure and layout.
- **CSS3**: For styling and responsiveness.
- **JavaScript**: For audio control and DOM manipulation.

## How to Run

1.  Clone this repository or download the source code.
2.  Ensure you have the audio files in the `songs` directory.
3.  Open `index.html` in your preferred web browser.
